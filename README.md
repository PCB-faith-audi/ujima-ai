# 🌍 Ujima AI — Ethical Financial Support System

Ujima AI is a multi-agent financial support platform designed to improve access to fair and transparent financial services for informal traders and SACCO members across East Africa.

It combines AI-driven analysis with **human oversight** to ensure ethical, explainable, and responsible financial decision-making.

## 🧠 Project Vision

Many informal traders struggle to access fair credit due to lack of formal financial records.

**Ujima AI solves this by:**
- Understanding seasonal income patterns (harvest cycles)
- Assessing loan risk fairly and transparently
- Providing financial literacy support
- Ensuring every critical decision is reviewed by a human officer

---

## 🏗️ System Architecture

Ujima AI is built around a **3-layer agent system**:

### 🌱 Scout Agent
- Analyzes income patterns and trading behavior
- Educates users on financial readiness
- Promotes savings habits and literacy

### 🧮 Guardian Agent
- Evaluates loan applications
- Assesses risk using income stability and seasonality
- Provides human-readable explanations (not just scores)

### 🤝 Hunter Layer (Human Review)
- Handles edge cases beyond AI confidence
- Ensures ethical and contextual decision-making
- Matches users with appropriate human loan officers

---

## ⚖️ Ethics & Compliance

Ujima AI is built on strict ethical principles:

- ✔ Human-in-the-loop decision making  
- ✔ Transparent AI explanations  
- ✔ No fully automated loan rejection/approval  
- ✔ Compliance with **Kenya Data Protection Act (DPA 2022)**  
- ✔ Alignment with SACCO regulatory frameworks (SASRA guidelines)
